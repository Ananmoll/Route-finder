# Route-finder

Route Finder is a Flask web app that helps you find routes between two places and view them on an interactive map.  
It also lets you connect your Spotify account to view your playlists while you travel üéß

## üöÄ Features

- Find driving routes between two locations  
- Use your **current location** or enter manually  
- Shows **distance and travel time**  
- Interactive **Folium map** with auto-zoom  
- Connect **Spotify** to view playlists and user profile
